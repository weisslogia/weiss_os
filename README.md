# Weiss_OS
## _proof of concept with c++_

# Resources
- [Write your own Operating System](https://www.youtube.com/@writeyourownoperatingsystem)

### Requirements packages

```sh
sudo apt install g++ binutils libc6-dev-i386 VIrtualBox xorriso
```
#### Build the kernel
```sh
make mykernel.bin
```

#### Build the iso

```sh
make mykernel.iso
```

### Clean the structure
```sh
make clean
```
